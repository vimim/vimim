VimIM 是專為 vim 玩家開發的插件，可作為 vim 內嵌中文輸入法。
隻要您的 vim 能顯示中文，您就能用 vim 輸入搜索中文。
VimIM 把 vim 的中文編輯潛能，提升到一個全新的高度。

概述

一個文本文件，構成一個中文輸入法。插上聯網就可以雲輸入。
如果再加上一個文本文件，無需聯網照樣敲打中文，隨心所欲。

程式文件用的是 vim script, 天生跨平台（Windows, Mac, Linux）
詞庫文件就是自己的輸入法（全拼或雙拼，五筆，英文，DIY）

所謂安裝，就是把文本文件扔到 vim 的 plugin 目錄之下。

不加設置，不換模式，點熱鍵就可上漢字，是謂〖點石成金〗。
如果切換成中文輸入模式，也可以將就當前流行的輸入法習慣。

所謂卸載，就是刪除文本文件。

Unicode 內碼輸入

Unicode 內碼輸入

Unicode 使漢字成為一種理想的電腦語言，沒有國界。
用 VimIM 玩萬國碼的前提是 :set encoding=utf-8
［科普］ http://zh.wikipedia.org/wiki/Unicode

萬國碼輸入器

  • 【廣告】精巧細致的標准Unicode輸入器，獨此一家。
  • 【理想】書同文，文同碼，碼同鍵。
  • 【特點】與輸入法和碼表都無關：以不變應萬變。
  • 【用途】內碼輸入是輸入非常用字符的有效途徑。
  • 【特點】一個字符對應一個代碼，武林高手可以中文盲打。
  • 【TIP】 UNICODE 內碼一次性列表　　:call CJK()

【必要條件】

  1. 入‘雲’必須聯網
  2. 入‘雲’必須借助 wget
  3. 入‘雲’必須使用漢語拼音：全拼或者雙拼
  4. 入‘雲’可能需要設置選項： g:vimim_www_sogou

【VimIM 雲輸入】

㈠ 〖混合式雲輸入〗設置 :let g:vimim_www_sogou=10
—— （長的）詩詞曲賦、小說、歌詞、完整的句子拜托給‘雲’。
—— （短的）私人信息、英文、詞組、單字留給VimIM自己處理。
—— 長短的定義沒有統一的國際標准。玩家自己定標准。
—— 例如：如果選項設置為10:　即每敲10個字母以上就交給‘雲’。

㈡ 〖整體性雲輸入〗設置 :let g:vimim_www_sogou=1
—— 每一個輸入都‘雲’游：

山寨雙碼輸入

用 VimIM 可使多種輸入方式和諧並存，互不沖突而又相互補充。
VimIM 充分發揮個人的知識潛能，達到“人盡其能、物盡其用”的境界。
  • 【用途】單碼（音碼或形碼）為主，雙碼（音碼加形碼）為輔。
  • 【單碼特點】二個單碼可混合使用，不用切換。
  • 【雙碼特點】音碼部分可以不精確。形碼部分可以不完整。
  • 【混合特點】雙碼優勢互補，各盡所能。
  • 【優點】用自己熟悉的輸入方式，用自己定的規矩：怎麼混也不亂！
  • 【結論】最好的輸入法是自己的輸入法。

【主要問題】
經典音碼：一音多字，一字多音。
流行形碼：難學難記，萬碼奔騰。
現代混碼：難記難學，各自為戰。

【解決方案】
【前提】模模糊糊知道兩種中文輸入方式，但都不精通。
　　　　第一步：尋找自己有點熟悉的輸入方式
　　　　第二步：結合與之相應的輸入法碼表
　　　　第三步：按照自己定的規矩：想怎麼打，就怎麼打！
【答案】模糊 + 模糊 = 精確
【原因】VimIM 幫您轉換碼表裡沒有，但您心裡有的輸入碼。

【理論基礎】
【通俗】負負得正
【學術】據說來自近代數學模糊集合理論

［科普］如何讓Vim支持中文？

  1. Vim 支持中文編碼的基礎

    • Vim 支持中文編碼的基礎是兩個特性：+multi_byte 和 +iconv
    • 官方下載網址： http://www.vim.org/download.php
    • 綠色移動版： http://portablegvim.sourceforge.net/downloads.html

  2. 影響 Vim 中文編碼的設置項

    • encoding：vim 的內部使用編碼
    • fileencodings：Vim 在打開文件時會根據其選項來識別文件編碼
    • fileencoding：Vim 在保存新建文件時會根據其設置編碼來保存
    • guifontwide：Gvim 要求等寬字體，其指定字寬是 guifont 的兩倍

  3. Vim 具體應用環境的設置：微軟視窗（Windows） vimrc 樣例

    • set encoding=utf-8
    • set fileencodings=ucs-bom,utf8,chinese,taiwan,ansi
    • set guifont=Courier_New:h12:w7
    • set guifontwide=NSimSun-18030,NSimSun
    • set ambiwidth=double

  4. 如何用 Vim 插入非 ASCII 字符？

    如果既不依賴外挂程式，又不干擾 Vim 正常操作，
    Vim 可以選擇如下三種方式，插入不能用普通鍵盤輸入的字符：
    1. 用 i_CTRL-V 插入十進制編碼
    2. 用 i_CTRL-K 插入二合字母
    3. 用 i_CTRL-6 〖點石成金〗 （VimIM 推陳出新）
